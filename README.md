# Quick Weather View

Quick Weather View는 날씨를 이미지로 보여주어 날씨를 빠르게 파악할 수 있도록 도와줍니다.

## How it works

사용자의 현재 위치와 시간을 기반으로 날씨 정보를 얻고 날씨에 맞는 이미지를 보여줍니다.
